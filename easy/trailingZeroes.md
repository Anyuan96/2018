# 问题分析

给定一个整数 n，返回 n! 结果尾数中零的数量。

	示例 1:
	
	输入: 3
	输出: 0
	解释: 3! = 6, 尾数中没有零。
	示例 2:
	
	输入: 5
	输出: 1
	解释: 5! = 120, 尾数中有 1 个零.
说明: 你算法的时间复杂度应为 O(log n) 。
# 代码实现
```C
int trailingZeroes(int n) {
    if (n/5 <5) return n/5;
    return n/5 + trailingZeroes(n/5);
}
```
# 总结体会
其实本题是求阶乘中有多少个10因式，也就是可以由多少个10组成，我们发现除了整十数之外，2x5也可以组成10，并且可以发现2因式比5因式多，整十数也可以拆成2x5，因此只要找出5因式的个数就可以了。
在刚做这道题的时候，我用的方法是现将阶乘的结果算出来，然后再计算0的个数，但是当输入的数值非常大时就会产生错误。
